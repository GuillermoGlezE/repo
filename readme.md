#BlackJack Vite

Pasos para ejecutar proyectos

1. Clonar repositorios
2. Ejecutar ```npm install``` reconstruir los modulos de node
3. Correr el devServer ```npm run dev ```
4. abrir el prooyecto en : localhost:3000 ```localhost:3000 ```
